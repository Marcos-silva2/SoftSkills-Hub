## 📇 Estrutura dos Cards (Frente e Verso)

Abaixo estão os termos sugeridos para o seu componente de cards interativos.

### 1. Feedback
* **Frente (O Termo):** 🗣️ Feedback
* **Verso (A Explicação):** É o famoso "retorno". É quando alguém avalia o que você fez para dizer o que foi bom e o que pode melhorar. **Dica:** Não encare como bronca, mas como um mapa para você crescer!

### 2. Deadline
* **Frente (O Termo):** ⏰ Deadline
* **Verso (A Explicação):** É a "linha de chegada" ou a data limite para entregar uma tarefa. **Dica:** Respeitar o deadline é a forma mais rápida de mostrar que você é um profissional de confiança.

### 3. Networking
* **Frente (O Termo):** 🤝 Networking
* **Verso (A Explicação):** É a sua rede de contatos profissionais. Conversar com pessoas de outras áreas e manter boas relações. **Dica:** O networking pode abrir portas para futuras promoções ou novas vagas.

### 4. Brainstorming
* **Frente (O Termo):** 🌩️ Brainstorming
* **Verso (A Explicação):** Significa "tempestade de ideias". É uma reunião onde todos dão sugestões livremente para resolver um problema. **Dica:** No brainstorming, não existe ideia burra. O objetivo é a criatividade!

### 5. Onboarding
* **Frente (O Termo):** 🚪 Onboarding
* **Verso (A Explicação):** É o processo de integração. É quando você entra na empresa e aprende sobre a cultura, as ferramentas e quem é quem na equipe.

### 6. Briefing
* **Frente (O Termo):** 📋 Briefing
* **Verso (A Explicação):** É o conjunto de instruções iniciais para realizar uma tarefa. **Dica:** Sempre leia o briefing duas vezes antes de começar para evitar retrabalho.

### 7. Soft Skills
* **Frente (O Termo):** 🧠 Soft Skills
* **Verso (A Explicação):** São suas habilidades comportamentais (como empatia, paciência e trabalho em equipe). São diferentes das "Hard Skills" (conhecimento técnico, como saber usar o Power BI).

### 8. Mentoria
* **Frente (O Termo):** 🎓 Mentoria
* **Verso (A Explicação):** É quando um profissional mais experiente orienta você na sua carreira. É uma oportunidade de ouro para aprender com os erros e acertos de quem já chegou lá.

## 🎨 Dicas de Design para o Style Guide

Para manter a consistência com o **SoftSkills Hub**:

* **Cor do Card:** Use a variável `--roxo` (ou um gradiente dela) para o fundo do card.
* **Animação:** Utilize `transform: rotateY(180deg)` no CSS para o efeito de virar.
* **Acessibilidade:** Garanta que o texto no verso tenha contraste suficiente (texto branco sobre o fundo roxo passa no teste WCAG AA).
* **Interação:** Como o foco é mobile-first, o card deve virar com um **toque (click)** simples, e não apenas com o hover do mouse.
